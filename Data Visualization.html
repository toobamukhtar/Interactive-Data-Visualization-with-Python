It seems like a pretty plot with all the colors and circles right? You might have noticed that why are there 2 bubbles with the same color? Well if you look back at the data then we grouped it by the sex column hence one circle is for females and 1 is for males.
What information can you takeaway from this plot?

  • There is a difference between gdp_per_capita for suicides/100k population between continents and between males and females, right?
  • To compare the values we will have to navigate between the x-axis and y-axis back and forth right? What if there was some way to display that information in the plot?
In [115]:
px.scatter(suicides_2009,x = 'suicides/100k pop', y = 'gdp_per_capita ($)',color='continent',size ='suicides/100k pop', title = 'GDP Per Capita for Suicides/100k Population (2009)')
  • Now the information is presented in a much better way. It's easier to read. Won't take that long that a small bubble represents a smaller value while a larger bubble represents a bigger value.
  • I can also hover over the bubbles to see the exact values rather then trying to approximate from the axis.
  • There is still something confusing about the plot. What if i want to instantly compare the values using the sex variable for the continents? There is no way to know using the plot.
  • To cater to this problem, we can use different symbols for males and females.
In [116]:
px.scatter(suicides_2009,x = 'suicides/100k pop', y = 'gdp_per_capita ($)', size = 'suicides/100k pop', color='continent',
           symbol='sex',title = 'GDP Per Capita for Suicides/100k Population (2009)'
        )

Now I can use my legend to distinguish between males and females. Take a moment to appreciate that how a little detail can add so much context to the plot!

  • There is a significant difference between the suicide rates of Male vs Females.
  • Asia has a lower GDP per capita and a lower suicide rate than Europe. Africa has the lowest GDP and the lowest suicide rate
  • The sucide rate for european males was the highest for the year 2009.
